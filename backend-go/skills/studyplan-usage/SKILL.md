# 学习计划技能指南 (StudyPlan Usage)

## 技能说明

本技能专注于管理宏观的学习路线和计划文档，与任务管理（PlanTask）不同，它处理的是整份 Markdown 文档。

## 核心操作

- **保存计划**：将生成的完整学习路线永久化存储。
- **读取计划**：召回之前的学习进度和规划，确保教学的连贯性。
- **版本管理**：支持对计划进行迭代和优化。

## 区分点

- **PlanTask**：具体的番茄钟、打卡项（JSON）。
- **StudyPlan**：完整的知识体系规划、学习目标（Markdown）。
