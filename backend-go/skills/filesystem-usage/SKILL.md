# 文件系统技能指南 (FileSystem Usage)

## 技能说明

本技能赋予 AI 读写本地文件和执行脚本的能力，支持更复杂的学习任务（如数据分析、笔记归档）。

## 核心操作

- **读取文件**：获取文件内容以供分析。
- **写入文件**：保存生成的笔记、代码段或计划。
- **执行环境**：可以运行特定脚本（如 Python）来辅助计算或处理数据。

## 安全与规范

1. **隔离性**：文件操作应严格限制在当前会话的工作目录内。
2. **确认性**：在写入重要文件前，应先向用户展示内容并获得确认。
3. **备份**：修改已有文件前，应考虑保留原始逻辑。
